**Hisabi** (হিসাবি — "accountant" in Bengali) is a fully offline, privacy-first personal finance management app built specifically for the Bangladeshi market. Track income & expenses, manage debts, plan budgets, create shopping lists, set savings goals, and generate PDF reports — all without an internet connection.

---

## ✨ Features

### 💰 Dashboard
- **Net Worth Card** — Real-time balance snapshot with performance indicator
- **Income / Expense / Savings** — Side-by-side monthly summary
- **Quick Actions** — One-tap access to Income, Expense, Transfer & Bazar
- **Recent Transactions** — Last 5 entries with category and account
- **Top Spending** — Horizontal bars for top expense categories
- **Upcoming Recurring** — Preview of scheduled transactions

### 📋 Transactions
- **Income / Expense / Transfer** — Full CRUD with category & account selection
- **Recurring Rules** — Daily, weekly, monthly, yearly schedules
- **Filter by Type** — All, Income, Expense, or Transfer views
- **Swipe to Delete** — With undo support

### 🏦 Accounts
- **18 Bangladeshi Banks** — Sonali, DBBL, BRAC, Islami Bank, EBL, and more
- **8 Mobile Wallets** — bKash, Nagad, Rocket, Upay, MyCash, SureCash, OK Wallet, Tap
- **9 Account Types** — Savings, Current, Mudaraba Savings, Cash on Hand, Piggy Bank, DPS, FDR, Mobile Banking, Custom
- **Opening Balance** — Track balance history from day one

### 🛒 Bazar (Shopping) Lists
- Create & manage shopping lists with 18 localized units (kg, প্যাক, বস্তা, etc.)
- Check off items and track actual vs. estimated prices
- **Convert to Expense** — Turn completed lists directly into transactions
- **Duplicate & Templates** — Reuse lists week after week
- **PDF Export** — Itemized shopping report

### 💳 Debts
- **I Owe / I'm Owed** — Dual-tab management
- **Payment Recording** — Partial/full payments auto-create expense transactions
- **Status Tracking** — Pending → Partial → Cleared
- **PDF Export** — Complete debt summary report

### 📊 Reports
- **Monthly Bar Chart** — Income vs. expense comparison
- **Expense Pie Chart** — Breakdown by category
- **Account Balances** — Quick reference
- **PDF Export** — Full financial report with date range filter

### 🎯 Savings Goals
- Set targets with deadlines, link to accounts
- Circular progress indicators
- **Deposit** directly toward goals

### 📆 Budgets
- Monthly budgets per expense category
- Visual progress bars (green / orange / red thresholds)
- Month-by-month navigation

### 🔒 Security
- 4-digit App PIN lock
- Encrypted storage via shared_preferences

### ⚙️ Settings
- Dark mode & accent color customization
- Currency selection (13 currencies)
- Backup & restore database
- Recurring transaction management
- Reset all data
